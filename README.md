# The Minimax Algo for TicTacToe implemented in Swift

## Eval

-1 Defeat

0  Tie

1  Win

Steps
1. Eval the current position
2. Recursively find all move trees
3. Eval trees and cut the losing ones
4. Determine the best trees
5. Make a move
